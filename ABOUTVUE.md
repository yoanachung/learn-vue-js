## Vue.js란?
MVVM 패턴의 뷰모델 레이어에 해당하는 화면단 라이브러리  
사용자 < DOM < View  <----> 뷰모델 (DOM Listners, Data Bindings) <----> 모델 (JavaScript)


## 뷰 컴포넌트
화면의 영역을 구분하여 개발할 수 있는 뷰의 기능  
재사용성을 늘려 빠르게 화면을 제작하기 위함이다.   
- 인스턴스를 생성하면 개발자도구에서 Root 컴포넌트로 인식한다.
